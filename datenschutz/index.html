Datenschutz auf programmicha.de

vom 30.12.2016

Es folgt eine Datenschutzerklärung.

Erhebung und Verarbeitung von Daten

Auf programmicha.de erhebe ich aktiv keinerlei Daten der Nutzer, hier gibt es also kein Google Analytics, Facebook Button, keine Cookies oder ähnliches.

Server (IP) Logging

Ich wollte auch das Server (IP) Logging komplett abschalten. Das geht bei meinem Webhoster Hosteurope nur eingeschränkt. Es wird aber eine Option zur Anonymisierung angeboten:

Ein weiteres Feature ist die Einstellung "Anonymisiert", die bewirkt, dass alle zugreifenden IPs im Logfile pauschal mit 127.0.0.1 gespeichert werden, und somit keine Rückschlüsse mehr auf die Besucher möglich sind.

— Hosteurope FAQ

Dies habe ich aktiviert. Immerhin etwas, inwiefern Hosteurope ggf. noch eigene Logs hält ist mir nicht bekannt. Eine kurze Anfrage direkt bei Hosteurope, ergab leider keine erschöpfende Aussage.

In Zeiten der Vorrats­datens­peicherung erwarte ich hier aber auch keine endgültige Anonymität.

Nutzung und Weitergabe personenbezogener Daten

Das ist einfach, ich erhebe einfach keine Daten die ich nutzen oder gar weitergeben könnte.

Gemacht aus und mit viel Neugier
