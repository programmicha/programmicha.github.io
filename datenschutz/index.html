Datenschutz auf programmicha.de

vom 30.12.2016

Es folgt meine Datenschutzerklärung für programmicha.de.

Erhebung und Verarbeitung von Daten

Auf programmicha.de erhebe ich keinerlei Daten der Nutzer, hier gibt es also kein Google Analytics, Facebook Button, keine Cookies oder ähnliches.

Die Seite selbst wird technisch bereitgestellt via Hosteurope, die als Domainverwalter auftreten, die Inhalte sind statisch bereitgestellt mit Github.

Hosteurope Server (IP) Logging

Eigentlich sollte keinerlei Traffic über die Server von Hosteurope laufen, da die Inhalte letztlich über DNS Umleitung direkt von Github kommen. Trotzdem habe ich für das gesamte Webhost Produkt nach Möglichkeiten für mehr Datenschutz gesucht und auch gefunden.

Es wird eine Option zur Anonymisierung im Server Log angeboten:

Die Einstellung "Anonymisiert" bewirkt, dass alle zugreifenden IPs im Logfile pauschal mit 127.0.0.1 gespeichert werden und somit keine Rückschlüsse mehr auf die Besucher möglich sind.

— Hosteurope FAQ

Diese habe ich aktiviert. Immerhin etwas, inwiefern Hosteurope ggf. noch eigene Logs hält ist mir nicht bekannt. Eine kurze Anfrage direkt bei Hosteurope, ergab leider keine erschöpfende Aussage.

In Zeiten der Vorrats­datens­peicherung erwarte ich hier aber auch keine endgültige Anonymität.

Datenschutz bei Github

Dazu gibt es eine Art Datenschutzerklärung bei Github: GitHub Privacy Statement . Der besondere Fall, also die Nutzung von Github Pages wird dort leider nicht direkt erwähnt.

Mit Stand 30.12.2016 kann ich aber sagen, dass keine Cookies reingestreut werden - wäre auch noch schöner, entspräche es doch einer Veränderung der Inhalte.

Nutzung und Weitergabe personenbezogener Daten

Das ist einfach, ich erhebe einfach keine Daten die ich nutzen oder gar weitergeben könnte.

Auskunftsrecht

Über die Kontaktdaten im Impressum kann man mich erreichen und Fragen zum Datenschutz (und natürlich alles andere) auf programmicha.de stellen.

Gemacht aus und mit viel Neugier
